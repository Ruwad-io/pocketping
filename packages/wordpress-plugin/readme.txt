Add a live chat widget to your WordPress site and reply to visitors from Telegram, Discord, or Slack.

== Description ==

PocketPing lets you chat with your website visitors directly from Telegram, Discord, or Slack - no need to keep a browser tab open.

**Features:**

* Easy setup - just paste your Project ID
* Reply from Telegram, Discord, or Slack
* Customizable colors to match your brand
* Mobile-friendly widget
* No account required on your site
* GDPR compliant

**How it works:**

1. Create a free account at [pocketping.io](https://pocketping.io)
2. Connect your Telegram, Discord, or Slack
3. Install this plugin and enter your Project ID
4. Start chatting with visitors!

**Perfect for:**

* Small businesses
* Bloggers
* E-commerce stores
* SaaS products
* Freelancers

== Installation ==

1. Upload the `pocketping` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > PocketPing
4. Enter your Project ID from [app.pocketping.io](https://app.pocketping.io)
5. Save and you're done!

== Frequently Asked Questions ==

= Where do I get a Project ID? =

Sign up for a free account at [pocketping.io](https://pocketping.io), create a project, and you'll find your Project ID in Settings > Installation.

= Is it free? =

PocketPing offers a free plan with 100 sessions/month. Paid plans are available for higher volume.

= Which messaging apps are supported? =

You can receive and reply to messages via Telegram, Discord, or Slack.

= Does it work with WooCommerce? =

Yes! The widget works on all pages including WooCommerce shop and checkout pages.

= Is it GDPR compliant? =

Yes, PocketPing does not use cookies and only stores necessary session data.
